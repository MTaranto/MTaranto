## Featured projects

### [Gem Bridge](https://github.com/MTaranto/gem-bridge)
A local-first daemon written in Go that provides AI assistants with controlled access to files, Git and development tools. The project includes workspace isolation, path validation, protection against path traversal and symlink attacks, explicitly allowed operations, tests and cross-platform continuous integration.

### [Gem Extension](https://github.com/MTaranto/gem-extension)
A browser extension that complements Gem Bridge through Native Messaging. The project explores secure communication between the browser and the local environment, minimal permissions and human confirmation before sensitive operations.

### [ListaRenata](https://github.com/MTaranto/ListaRenata)
A web application created to solve a real need: managing a gift list and reducing duplicate choices. It uses HTML, JavaScript, Tailwind CSS and Supabase/PostgreSQL, with real-time data reading and updates.

### [Parking Management System — Go + HTMX](https://github.com/MTaranto/estacionamento-go-htmx)
A local system under development for managing vehicle entry, stay and exit. It uses Go, HTMX, server-side templates and SQLite, with capacity rules, license plate validation and dynamic interface updates.

### [AI Persona Architecture](https://github.com/MTaranto/arquitetura-de-persona-ia)
A case study on context engineering, system prompting and the organization of long-running interactions with language models. The repository documents hypotheses, instruction patterns and lessons learned through practical experiments.
